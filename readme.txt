== Description ==

A customizable widget which displays the latest related Games Tag based from AOL Games. It can be integrated anywhere in the blog. This widget shows up the lastest five Games related to tags placed on posts from the worlds most popular Gaming site 
http://www.games.com/. This is a very nice solution for all those wanting to share latest related games.

*Feature List*

* customizable widget
* displays a user-defined number of links
* title shortening

== Installation ==

*Could not be simpler*
1. Extract `related-games.zip`.
2. Upload the `games-rss-plugin`-folder to your `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Add the widget to your sidebar from Appearance->Widgets and configure the widget options.
